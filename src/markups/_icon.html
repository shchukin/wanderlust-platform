.icon

Icons come from a Material Symbols Rounded font. Note, it is not the default icon font (like on the promo site), but it is Rounded variation.

Use any tage with class: .icon, like so:

grid_view cycle star

See the list of available icons on https://fonts.google.com/icons



Set up

Generate a string with all necessary icons like this:

https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:opsz,wght,FILL,GRAD@24,400..700,0,0&icon_names=add,alarm_pause,arrow_upload_progress,bookmark,brand_awareness,camera,check_box,check_box_outline_blank,close,cloud,contact_support,content_copy,crown,cycle,delete,diamond_shine,done_all,draft_orders,east,edit,filter_list,graphic_eq,grid_view,handshake,keyboard_arrow_down,keyboard_double_arrow_left,keyboard_double_arrow_right,logout,mail,more_vert,new_label,notifications_active,phone,play_arrow,radio_button_checked,radio_button_unchecked,redeem,rotate_right,savings,settings,star,supervisor_account,thumb_down,thumb_up,visibility,visibility_off,west

Add this file to the project as a normal CSS file or save generated woff2 font file into fonts/ folder and write CSS manually in the styles/global/fonts as it was set up for now.
