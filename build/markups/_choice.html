.choice--full-width

By default, the .choice component has a width based on the text inside (fit-content). However, in some cases, we want it to be full-width, for example, in a list of options in the sidebar or in a dropdown. In this case, we want to move the cursor over all checkboxes and be able to click them even if some of them are shorter.
