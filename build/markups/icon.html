.icon

Icons come from a Material Symbols Rounded font. Note, it is not the default icon font (like on the promo site), but it is Rounded variation.

Create a component with: .icon class like so:

grid_view cycle star

(any tag will do)

Color can be set via color css property (like a normal font). Width and height can be set via font-size and line-height (use the same value).

Example: 

.callout__close .icon {
    font-size: 24px;
    line-height: 24px;
    color: #5c6690;
}


Set up

See the list of available icons on https://fonts.google.com/icons

Generate a string with all necessary icons to address google font like this:

<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:opsz,wght,FILL,GRAD@24,400..700,0,0&icon_names=add,alarm_pause,arrow_upload_progress,bookmark,brand_awareness,camera,check_box,check_box_outline_blank,close,cloud,contact_support,content_copy,crown,cycle,delete,diamond_shine,done_all,draft_orders,east,edit,filter_list,graphic_eq,grid_view,handshake,keyboard_arrow_down,keyboard_double_arrow_left,keyboard_double_arrow_right,logout,mail,more_vert,new_label,notifications_active,phone,play_arrow,radio_button_checked,radio_button_unchecked,redeem,rotate_right,savings,settings,star,supervisor_account,thumb_down,thumb_up,visibility,visibility_off,west"/>

Either add this to the <head> or open the link and save the contains of that file into to the fonts/ folder together with generated woff2 file.
