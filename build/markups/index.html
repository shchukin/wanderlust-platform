Components Documentation

Welcome to the project documentation. Here you can find examples and usage instructions for various UI components and libraries used in this project.



Select & Autocomplete Libraries

We use several libraries for enhanced selection components. Choose a library to see its examples and configuration options:

  • Tom Select — A flexible selection component with support for autocomplete, tagging, and custom rendering.
  • Choices.js — A lightweight, configurable select box/text input plugin (no jQuery dependency).

Other Components

Standard UI components and utility examples:

  • SweetAlert2 — Beautiful, responsive, customizable replacement for JavaScript's popup boxes.
  • Buttons — Standard button styles and variations.
  • Inputs — Form input fields and controls.
  • Avatars — User profile images and placeholders.
